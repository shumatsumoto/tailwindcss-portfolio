## :pencil2: Overview

Learn Tailwindcss while creating a personal portfolio site.

## :hammer: Requirement

- macOS
- Tailwindcss 3.0

## :pushpin: Usage

Git clone
```
$ git clone https://github.com/shumatsumoto/tailwindcss-portfolio.git
```
Move to directory
```
$ cd tailwindcss-portfolio
```
Install the library
```
$ npm install
```
Launch a local server
```
$ npm run build
```

Access URL： 
http://localhost:5500
（Live Server）
